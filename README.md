# South Texas Roofing — Marketing Website

Production-ready single-page marketing site built with **Next.js 16** (App Router,
RSC), **Tailwind CSS v4**, **Framer Motion**, and **lucide-react**. Replicates the
"TitanPipe" Framer layout with the company's real content.

## Quick start

```bash
npm install
node scripts/gen-placeholders.mjs   # (already run) creates /public/images placeholders
npm run dev                          # http://localhost:3000
npm run build && npm start           # production
```

## Editing content & swapping assets — one file

**`lib/site-config.ts`** is the single source of truth. All copy, nav links,
services, FAQ, testimonials, NAP data, and **every image/video path** live here.

To drop in the real logo and photography:

1. Place your files in `public/images/` (keep the placeholder filenames, e.g.
   `logo.svg`, `hero-poster.jpg`, `about-1.jpg`, …), **or**
2. Update the matching path in the `images` map in `lib/site-config.ts`.

For a real hero background video, add `public/images/hero-roof.mp4` (already
wired in `images.hero.video`); until then the poster image shows.

The placeholder assets are generated by `scripts/gen-placeholders.mjs`.

## Structure

```
app/
  layout.tsx        Inter Tight font, SEO metadata, OG, JSON-LD LocalBusiness
  page.tsx          Section composition
  globals.css       Tailwind v4 @theme tokens + component utilities
components/          One file per section
  Header  Hero  RequestForm  About  Projects  TrustFeatures
  ServicesSlider  Process  Materials  MidCta  Testimonials
  Faq  FinalCta  Footer  Marquee  Reveal
lib/site-config.ts  ALL content + asset paths
public/images/      Placeholder SVGs (swap for real assets)
scripts/            Placeholder generator
```

## Design tokens (`app/globals.css` → `@theme`)

| Token | Value | Use |
|-------|-------|-----|
| `--color-yellow` | `#F2C84B` | accent, CTAs, stat card |
| `--color-ink` | `#1C1C1C` | headings / text on light |
| `--color-black` | `#0B0B0B` | footer, dark bands |
| `--color-muted` | `#6E6E6E` | body copy |
| `--color-bg` | `#FAFAF8` | page background |
| `--color-card` | `#F1F1EF` | light panels |

## Notes

- The hero **Request Services** form and footer newsletter are client components
  with local state and a `console.log` submit handler — each marked
  `// TODO: wire to endpoint`. No backend is included.
- Slider, testimonials carousel, and FAQ accordion are keyboard-navigable with
  proper ARIA roles; animations respect `prefers-reduced-motion`.
- SEO: per-route metadata, Open Graph/Twitter cards, and JSON-LD
  `RoofingContractor` schema using the real NAP data.
